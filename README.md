# 究極の運勢占い ✨ - Google Gemini AI版

8種類の占術で完全鑑定し、総合運勢をAIが自動生成します!

## 🎯 特徴

- **8種類の占術**: 九星気学、数秘術、四柱推命、西洋占星術、五星三心、カバラ、紫微斗数、タロット
- **AI生成の総合運勢**: Google Gemini AIが毎回ユニークな文章を生成
- **完全無料**: Google Gemini APIの無料枠を使用(月60リクエストまで無料)

## 🚀 セットアップ方法

### 1. Google Gemini APIキーの取得

1. [Google AI Studio](https://makersuite.google.com/app/apikey) にアクセス
2. Googleアカウントでログイン
3. 「Create API Key」ボタンをクリック
4. APIキーをコピー

### 2. APIキーの設定

`script.js` ファイルを開き、**543行目付近**の以下の部分を編集:

```javascript
// ★★★ ここにあなたのGoogle Gemini APIキーを入力してください ★★★
const GEMINI_API_KEY = 'YOUR_API_KEY_HERE';
```

↓ 変更後

```javascript
// ★★★ ここにあなたのGoogle Gemini APIキーを入力してください ★★★
const GEMINI_API_KEY = 'あなたのAPIキーをここに貼り付け';
```

### 3. ファイルをアップロード

GitHub Pagesなどの静的ホスティングサービスにアップロード:
- `index.html`
- `styles.css`
- `script.js` (APIキーを設定したもの)

### 4. 完了!

ブラウザでアクセスして占いを楽しんでください!

## 📝 使用方法

1. 生年月日を入力
2. (オプション) 生まれた時間を入力
3. (オプション) お名前を入力
4. 「占う!」ボタンをクリック
5. AIが総合運勢を自動生成します

## 🔒 セキュリティに関する注意

- APIキーは公開リポジトリにコミットしないでください
- プライベートリポジトリを使用するか、環境変数を使用することをお勧めします
- 不正使用を防ぐため、Google Cloud Consoleで使用制限を設定できます

## 💡 トラブルシューティング

### エラーが出る場合

1. ブラウザのコンソール(F12キー)を開いて詳細を確認
2. APIキーが正しく設定されているか確認
3. Google Gemini APIの無料枠を超えていないか確認

### APIの無料枠について

- **Gemini 1.5 Flash**: 月60リクエストまで無料
- それ以上使用する場合は、Google Cloudの課金設定が必要です

## 🎨 カスタマイズ

- `script.js`のプロンプト部分: AIの文章スタイルを変更
