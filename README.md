# 究極の運勢占い ✨

8種類の占術で完全鑑定する運勢占いウェブアプリケーション

## 🎯 特徴

- **8種類の占術で完全鑑定**
  - 九星気学
  - 数秘術
  - 四柱推命
  - 西洋占星術
  - 五星三心占い
  - カバラ占術
  - 紫微斗数
  - タロット(2026年運勢)

- **ダイナミックな総合運勢**
  - ランダム要素を組み合わせた多彩な文章
  - 毎回異なる表現で占い結果を提示

- **AI占いへの連携機能**
  - 占い結果をワンクリックでコピー
  - Gemini AIで詳しく占うボタン搭載

## 🚀 使い方

### 基本的な占い

1. 生年月日を入力(必須)
2. 生まれた時間を入力(オプション)
3. お名前を入力(オプション)
4. 「占う!」ボタンをクリック
5. 8種類の占術結果と総合運勢が表示されます

### Gemini AIで詳しく占う

1. 占い結果画面の下部にある「占い結果をコピー」セクションへスクロール
2. テキストボックスに占い結果が自動生成されています
3. 「📋 結果をコピー」ボタンをクリック
4. 「✨ Geminiで占う」ボタンをクリック
5. Gemini AIのページが開いたら、コピーした結果を貼り付け
6. AIが詳細な鑑定を提供します

## 🌐 デプロイ方法

### GitHub Pagesにデプロイ

1. GitHubに新しいリポジトリを作成
2. すべてのファイルをアップロード
3. Settings → Pages → Source を `main` ブランチに設定
4. 数分後、URLが生成されます

### その他のホスティング

- Netlify
- Vercel
- Cloudflare Pages

など、静的サイトホスティングサービスならどこでも動作します。

## 🎨 カスタマイズ

### デザインの変更

`styles.css`を編集してカラーやレイアウトを変更できます。

### 占い結果の文章パターン

`script.js`の`displayTotal`関数内にある配列(`openings`, `yearForecasts`, `elements`など)を編集することで、総合運勢の文章バリエーションをカスタマイズできます。

## 📋 占術の詳細

### 九星気学
生年月日から九星を算出し、性格傾向、ラッキーカラー、ラッキー方位を鑑定

### 数秘術
生年月日から運命数を計算し、人生の傾向を読み解く

### 四柱推命
年・月・日・時の四柱から五行バランスを分析

### 西洋占星術
生年月日から星座を判定し、性格や運勢を鑑定

### 五星三心占い
生年月日から12タイプを判定し、性格傾向を分析

### カバラ占術
数秘術と同じ運命数を使用し、スピリチュアルな観点から鑑定

### 紫微斗数
中国の伝統的占星術で主星を判定

### タロット
2026年の運勢を大アルカナ22枚から導き出す
