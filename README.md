# 究極の運勢占い ✨ - AI生成総合運勢版

8種類の占術で完全鑑定し、Google Gemini AIが毎回ユニークな総合運勢を自動生成します!

## 🎯 特徴

- **8種類の占術**: 九星気学、数秘術、四柱推命、西洋占星術、五星三心、カバラ、紫微斗数、タロット
- **AI生成の総合運勢**: Google Gemini AIが毎回ユニークな文章を生成
- **ブラウザ上でAPIキー設定**: 安全にパブリック公開できる設計
- **完全無料**: Google Gemini APIの無料枠を使用(月60リクエストまで無料)

## 🚀 セットアップ方法

### GitHubにアップロードするだけ!

1. このリポジトリをGitHubにアップロード
2. GitHub Pagesを有効化
3. 完了!

**ユーザーは自分でAPIキーを設定**するため、あなたのAPIキーを埋め込む必要はありません。

## 👤 ユーザー側の使用方法

### 1. 初回アクセス時

初めてサイトにアクセスすると、APIキー設定画面が自動的に表示されます。

### 2. Google Gemini APIキーの取得

1. [Google AI Studio](https://makersuite.google.com/app/apikey) にアクセス
2. Googleアカウントでログイン
3. 「Create API Key」ボタンをクリック
4. APIキーをコピー

### 3. APIキーの入力

- 設定画面にAPIキーを貼り付け
- 「保存」ボタンをクリック
- ブラウザのlocalStorageに安全に保存されます

### 4. 占いを実行

1. 生年月日を入力
2. (オプション) 生まれた時間を入力
3. (オプション) お名前を入力
4. 「占う!」ボタンをクリック
5. AIが総合運勢を自動生成します ✨

### 5. 設定の変更

右上の⚙️ボタンからいつでもAPIキーを変更できます。

## 🔒 セキュリティ

### なぜ安全なのか?

- ✅ APIキーはユーザーのブラウザにのみ保存(localStorage)
- ✅ サーバーやリポジトリには保存されない
- ✅ 各ユーザーが自分のAPIキーを使用
- ✅ あなたのAPIキーが消費されることはない

### ユーザーへの注意事項

- APIキーは他人と共有しないでください
- 公共のPCを使用する場合は、使用後にブラウザのデータをクリアしてください

### デザインの変更

## 🐛 トラブルシューティング

### 総合運勢でエラーが出る

1. ブラウザのコンソール(F12)でエラー内容を確認
2. APIキーが正しく入力されているか確認
3. APIキーが "AIza" で始まっているか確認
4. 月の無料枠を使い切っていないか確認

### APIキーを忘れた場合

ブラウザのlocalStorageをクリアするか、右上の⚙️ボタンから新しいAPIキーを設定してください。
