# ✨ トリプル運勢占い

九星気学 × 数秘術 × 四柱推命を組み合わせた本格的な占いサイト

## 🌟 特徴

- **九星気学**: 生年月日から本命星を算出し、ラッキーカラーと方位をお知らせ
- **数秘術**: 運命数を計算し、あなたの本質的な性格や才能を明らかに
- **四柱推命**: 年・月・日・時の四柱から五行のバランスを分析
- **ポップなデザイン**: 明るく親しみやすいUI/UXで楽しく占える
- **モバイル対応**: スマートフォンでも快適に利用可能

## 🚀 デモ

[GitHub Pagesでデモを見る](#) ※デプロイ後にURLを追加

## 📋 使い方

1. 生年月日を入力
2. （任意）生まれた時間を入力すると、より詳細な四柱推命の結果が得られます
3. （任意）お名前を入力すると、数秘術の計算に名前も反映されます
4. 「占う！」ボタンをクリック

## 🎨 占いの種類

### 九星気学
東洋の伝統的な占術。生年月日から9つの星（一白水星〜九紫火星）を算出し、その人の基本的な性格や運勢、相性、方位の吉凶を判断します。

### 数秘術
ピタゴラスが創始した占術。生年月日や名前から運命数を算出し、人生の目的や才能、課題を読み解きます。

### 四柱推命
中国発祥の最も複雑で詳細な占術の一つ。年・月・日・時の四つの柱から十干十二支を算出し、五行のバランスから運勢を詳細に分析します。

## 🔮 占いの精度について

このサイトは娯楽目的で作成されています。占い結果は一般的な解釈に基づいており、個人の運命を決定づけるものではありません。楽しみながらご利用ください。
